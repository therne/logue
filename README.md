Logue
====================
Logue is intention-based text understanding framework.  
Optimized for Korean, English.  **THIS PROJECT IS WORK IN PROGRESS - NOT USABLE.**

## Prerequisites
* [TensorFlow](https://tensorflow.org) >= 1.0
* [KoNLPy](http://konlpy.org) for preprocessing Korean 
    * Java Runtime >= 1.6
* Python >= 3.5

## Usage
You need to install dependencies first.
```
git clone https://github.com/therne/logue & cd logue
./install.sh
```

Then train the model using your own data:
```
./train.py WIP
```
